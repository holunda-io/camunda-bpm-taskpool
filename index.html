BPM Taskpool

A component for pooling user tasks with business data for high performance queries.

Download

  • Home

Why should I use this?

  • Are you building a process application or process platform containing multiple process applications using Camunda BPM Engine?
  • Are you building a custom task list for your existing application using Camunda BPM Engine?
  • Are you trying to solve performance issues with your task list?
  • Are you building an archive view for the business objects processed by your processes?
  • Are you interested in an audit log for any changes performed during the execution of the business process?

If you can answer one of the previous questions with yes, the taskpool / datapool library might help you.

How to start?

We provide documentation for different people and different tasks. A good starting point is the Introduction. You might want to look at User Guide containing a Working Example and details about Usage Scenarios.
